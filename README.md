# MyReadsApp
MyReads is a Book Shelf App for Readers to keep track of their reads and search books

Styling for this project has been primarily taken from the Udacity MyReads starter template along with the BooksAPI.

## Installation and Setup
1. Download or clone this repo
2. run `npm install`
3. run `npm start`
4. running step 3 should bring up the browser window to http://localhost:3000/

## Important
Please be aware that the API being used in the search page is limited to a specific amount of terms. These terms can be found on [SEARCH_TERMS.md](SEARCH_TERMS.md).
